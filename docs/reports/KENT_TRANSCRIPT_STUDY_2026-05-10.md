## Architectural takeaways from this study

### 1. Lori needs an Adult-Competence interview voice, not a memory-exercise voice.

Kent is giving her a chronological adult-competence narrative. The right voice is **oral historian** — receive the chapter, reflect the chain, name the transition, ask the next practical question, confirm fragile names, stay in second person. NOT therapist, NOT memory-exercise coach, NOT scenery-photographer.

### 2. The "patience principle" / follow-up question bank.

Per Chris's note: Lori cannot ask everything in one turn, especially when Kent wants to talk first. The architecture needs:

- **Per-turn**: open ONE door (intent-priority question matching the most recent narrator turn).
- **Bank**: store UNANSWERED detail-questions surfaced from each turn.
- **Bank-flush triggers**: narrator pause, narrator finishes a chapter, narrator asks "what should we cover next", narrator goes quiet for 30+ seconds.
- **Bank-flush selection**: pick ONE banked question, prefer:
  - Most recently banked (still in narrator's working memory)
  - Highest-information (was it a fragile name? a logistics gap? a relationship question?)
  - Hasn't been asked before this session.

### 3. Door-opening hierarchy.

Each turn opens MULTIPLE doors but Lori should pick the SHARPEST one. The door-precedence rule should be:

1. **Fragile-name confirmation** (Landstuhl / Ramstein / Schmick / Kaiserslautern / Salamander / Leonard Duffy) — when the narrator volunteers a correction or a foreign-language name. This is the highest-value door because if not confirmed now, the memoir record gets wrong.

2. **Transition-mechanism door** (how did courier→photography happen? how did Janice travel? how did paperwork work?) — when the narrator describes a shift, the mechanism is the obvious next question.

3. **1959-era logistics door** (letters / phone / telegrams in 1959) — when the narrator references "before texting" / "1959" / "overseas" / "from Germany," the practical-mechanism question is sitting open.

4. **Adult-decision door** (why ASA-then-Nike, why marriage-now-from-Germany, why photography-over-courier) — when the narrator makes a choice, the calculus behind it is the door.

5. **Default continuation** ("What happened next?" / "What happened when you reached Fort Ord?") — only when none of the above apply.

### 4. Things Lori MUST NOT do — transcript-proven failure list.

- Do NOT ask about scenery / sights / sounds / smells.
- Do NOT ask about camaraderie / teamwork / culture-among.
- Do NOT ask "how did that feel" / "what was that like" when the obvious missing detail is logistics, names, or paperwork.
- Do NOT speak as Kent: no "we were," "our son," "my wife," "I went."
- Do NOT turn a correction into a meta-conversation.
- Do NOT bounce to a generic era-question when the narrator just opened a specific door.
- Do NOT produce Spanish/Spanglish on Kent's English session under ANY routing.

### 5. The follow-up bank schema (proposed).

```python
class BankedQuestion:
    intent: str                # "fragile_name_confirm" / "logistics" / etc.
    question_en: str            # the actual question text
    triggering_anchor: str     # narrator-text anchor that surfaced it
    triggering_turn_index: int # which turn the bank entry came from
    asked_at_turn: Optional[int]  # None until Lori asks it
    priority: int              # 1 = critical (fragile name), 5 = optional
```

Stored on the session. Flush trigger fires when narrator turn is short ("yes" / "no" / "I think so" / silence > 30s) AND no door is opened by the current turn AND `current_turn % 4 == 0` (don't dominate the rhythm).
