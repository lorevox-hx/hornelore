# Research papers (local copies, gitignored)

This directory holds local copies of research papers cited in
`../references.md`. PDFs are gitignored so the repo doesn't carry
binary blobs, but the directory itself is tracked (via this README +
the .gitignore) so the layout is consistent across clones.

To populate locally:
- Drop PDFs here. They will not be committed.
- Add an entry in `../references.md` with the source URL / DOI and
  this local filename for offline reference.

If a paper has an open-access canonical URL, prefer linking to that
rather than relying on a local copy. The local copy is for offline
work; the canonical record lives in `references.md`.
